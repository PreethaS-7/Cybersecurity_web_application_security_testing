# Cybersecurity_web_application_security_testing

Objective

The objective of this task was to understand web application security testing concepts and perform security assessment activities in a controlled environment using OWASP Juice Shop.

Tools Used

- Kali Linux
- Docker
- OWASP Juice Shop
- Burp Suite Community Edition
- Dirsearch
- Firefox Browser

Activities Performed

- Setup and verification of OWASP Juice Shop
- HTTP traffic analysis using Burp Suite
- Request inspection and analysis
- Burp Repeater usage
- Directory discovery using Dirsearch
- Study of SQL Injection and Cross-Site Scripting concepts
- Documentation and reporting

Key Learnings

- Web Application Security Fundamentals
- HTTP Request and Response Analysis
- Burp Suite Workflow
- Directory Enumeration
- SQL Injection Concepts
- Cross-Site Scripting Concepts
- Penetration Testing Documentation

Outcome

Successfully completed Task 4: Exploitation & Web Application Security Testing in a controlled lab environment and documented the findings in a penetration testing report.
